# Pentobi 

[Pentobi](https://pentobi.sourceforge.io/) is a computer opponent for Blokus built for Linux.

This directory contains helps for running Pentobi in Docker. Execute ./run from within the `pentobi` directory to build a Docker image from the enclosed Dockerfile, start a container from the image, and open a browser with a VNC into the Docker container to play Pentobi.
